# Giuliana Sambon - UX Design Portfolio

Personal portfolio website showcasing UX/UI design work and case studies.

## Projects

### Contactabilidad — Communication Platform Redesign

A comprehensive case study on redesigning a communication platform to improve user engagement and contact accessibility. Features include:

- **Multi-channel communications** - Email, SMS, WhatsApp integration
- **Smart notifications** - Intelligent alert system
- **Intuitive interface** - Desktop and mobile optimized layouts
- **User-centered design** - Research-driven interaction patterns

View the full case study: [Contactabilidad UX Case Study](./index.html)
